Problem-shaped VM demo

Weiqi

A 5x5 capture-PSVM with a deliberately tiny op set. The worker solves a local capture problem exactly while the UI streams the search trace. The visible VM surface is just PLAY, CAPTURE, UNDO, PASS, and HALT; liberties, suicide, and ko remain deterministic engine semantics underneath.

    Why this shape
    
            
    
              Full 19x19 Go is the wrong first benchmark for a browser-local PSVM. A
              small-board local capture problem keeps the state exact and the instruction
              set honest, while still forcing the runtime to handle chain detection,
              capture, suicide, and ko correctly.
